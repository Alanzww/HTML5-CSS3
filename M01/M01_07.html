Segundo Jeff Noble, seu livro HTML para leigos:

HTML é a sigla para HyperText Markup Language, ou seja, Linguagem de Marcação de Hipertexto. O HTML é a linguagem de marcação usada para criar páginas da web. Ele é composto por uma série de elementos que descrevem a estrutura e o conteúdo de uma página, como títulos, parágrafos, links, imagens e muito mais. O HTML é a base da construção de sites e é essencial para qualquer pessoa que queira aprender a criar conteúdo para a web.
